# readme

## Bold

add "##" to bold

## hello world

## italics

add "_" after
hello world_

## bullet points

add "-" before content

- hello world

## quote

add ">" before content

> hello world

## link

add "[]" for content and "()" for link
[github](https://github.com) - link to github

## CLI

add "```sh" before and after (remove sh after)

```sh
yarn add typescript
```
